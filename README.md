# Check-Wi-Fi-connection-with-ESP32
This sytem permits users to check their wifi connection availability, the components needed is just an OLED screen(optional, you can use the serial monitor) and ESP32 or Arduino, it is super easy and simple.
if you are using the OLED then wire:
SCL to GPIO 22 
SDA to GPIO 21
VCC to 3V3
GND to GND
install WiFi, ESPping, SPI, Wire, Adafruit_SSD1306 and Adafruit_GFX libraries from the manage libraries in the Arduino IDE left bar.
